## Using Benchmark.js

# Run benchmarks from the commandline

Run with `npm test` or `karma start`.


# Run benchmarks in the browser

Just open `test/index-bjs.html`.


## Using JSLitmus

Just open `test/index-jsl.html`.


## Setup

# Benchmark.js

Dependencies:
- test/setup/benchmark.js
- package.json (to run in the command line)
- karma.conf.js (to run in the command line)
- test/setup/SimpleLogger.js (to run in the browser)
- test/setup/bootstrap.min.css (for nicer browser output)

Installation:
Run `npm install`.

# JSLitmus

Dependencies:
- JSLitmus.js
